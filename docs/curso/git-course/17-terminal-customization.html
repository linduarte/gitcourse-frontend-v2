Customização do Terminal com Oh My Posh

Personalizar o terminal não é apenas uma questão estética — é uma forma de tornar o ambiente de trabalho mais eficiente, claro e agradável. A seguir, apresentamos uma comparação direta entre o terminal padrão e o terminal customizado com Oh My Posh, destacando como essa mudança simples pode melhorar significativamente a experiência de desenvolvimento.

Antes — Terminal Padrão

O terminal tradicional do Windows cumpre sua função, mas oferece pouca informação visual. Sem cores, sem indicadores de status e sem contexto sobre o repositório Git, ele exige mais esforço para interpretar o estado atual do projeto.

  • Visual neutro e pouco informativo
  • Sem indicação da branch atual
  • Sem destaque para arquivos modificados
  • Dificuldade para identificar rapidamente o estado do repositório

Depois — Terminal com Oh My Posh

Com Oh My Posh, o terminal ganha vida: cores, ícones, informações úteis e um layout moderno que facilita o trabalho diário. A leitura fica mais clara, o status do Git aparece automaticamente e o ambiente se torna muito mais agradável.

  • Exibição clara da branch atual
  • Ícones e cores indicando status do Git
  • Prompt mais organizado e visualmente agradável
  • Melhor leitura e navegação entre projetos
  • Aparência moderna e profissional

Por que usar Oh My Posh?

Benefício Descrição
Produtividade Informações essenciais aparecem automaticamente no prompt.
Clareza Cores e ícones facilitam a leitura e reduzem erros.
Estética Um ambiente mais agradável melhora a experiência de uso.
Profissionalismo Ideal para apresentações, gravações e documentação.

1. Instalando uma Nerd Font (obrigatório)

Tanto Oh My Posh quanto Starship usam ícones especiais. Para que eles apareçam corretamente, você precisa instalar uma Nerd Font.

Recomendamos a fonte Caskaydia Cove Nerd Font.

winget install NerdFonts.CaskaydiaCove

Depois, abra o Windows Terminal → Configurações → Aparência → Fonte → selecione a Nerd Font instalada.

2. Instalando o Oh My Posh

O Oh My Posh é um dos prompts mais completos e bonitos disponíveis para Windows.

Instalação

winget install JanDeDobbeleer.OhMyPosh -s winget

Ativando no PowerShell

Edite seu perfil do PowerShell:

notepad $PROFILE

Adicione esta linha:

oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH\paradox.omp.json" | Invoke-Expression

Salve e abra um novo terminal.

3. Instalando o Starship (alternativa multiplataforma)

O Starship é rápido, leve e funciona em qualquer sistema operacional.

Instalação

winget install Starship.Starship

Ativando no PowerShell

notepad $PROFILE

Adicione:

Invoke-Expression (&starship init powershell)

4. Temas recomendados

  • Oh My Posh – Paradox (equilibrado e informativo)
  • Oh My Posh – Agnoster (clássico para Git)
  • Starship – Default (limpo e rápido)
  • Starship – Pure (minimalista)

Você pode testar outros temas com: 

oh-my-posh config list

5. Como o terminal ajuda no Git

  • Mostra a branch atual
  • Indica arquivos modificados
  • Mostra commits pendentes
  • Exibe status do repositório em tempo real
  • Mostra tempo de execução de comandos
  • Ajuda a evitar erros como commit na branch errada

Com o terminal customizado, você terá uma experiência muito mais fluida e profissional ao longo do curso.
