Seu Terminal, Seu Painel de Controle

Aprenda a instalar o Starship: o motor que transforma seu terminal em uma ferramenta profissional, rápida e totalmente em português.

Configurar Meu Terminal Agora

Customização Profissional com Starship

🌎 Um só comando, qualquer sistema: O Starship é multiplataforma. As instruções abaixo focam no Windows, mas se você usa Linux ou Mac, o visual e a configuração final serão exatamente os mesmos!

Para dominar o Git, você precisa de clareza. O Starship funciona como um GPS: ele te diz exatamente onde você está, em qual ramificação (branch) do projeto você trabalha e se o seu Python está pronto para o combate.

O Salto de Qualidade

Antes: Terminal Comum

Escuro, sem cores e sem informações sobre o seu progresso no Git.

Depois: Seu Novo Painel

Tudo em Português: Pasta atual, ícone do Python e status do Git em tempo real.

Guia de Instalação (Windows)

1. O Novo Terminal

Antes de tudo, instale o Windows Terminal. Ele permite abrir várias abas e mostra as cores corretamente.

winget install Microsoft.WindowsTerminal

2. A Fonte Especial (Nerd Font)

Para que os ícones (como a 🐍 do Python) apareçam, você precisa da fonte Fira Code Nerd Font.

winget install NerdFonts.FiraCode

Dica: Após instalar, abra o Windows Terminal, vá em Configurações > Aparência e mude a fonte para "FiraCode Nerd Font".

3. Instalando o Motor (Starship)

Agora, vamos instalar o motor que processa as informações:

winget install Starship.Starship

4. Ativando e Traduzindo

Abra o arquivo de configuração do seu terminal:

notepad $PROFILE

Adicione esta linha ao final do arquivo, salve e feche:

Invoke-Expression (&starship init powershell)

5. Configuração em Português

Para que o seu terminal fale a sua língua, crie o arquivo de configuração:

notepad ~/.config/starship.toml

Cole o código abaixo para ter um visual limpo e eficiente:

# Configuração simplificada para o Curso do Charles
add_newline = true

[character]
success_symbol = "[➜](bold green) "
error_symbol = "[✗](bold red) "

[directory]
style = "bold cyan"
placeholder = "Pasta Atual"

[git_branch]
symbol = " "
format = "em [$symbol$branch]($style) "
style = "bold magenta"

[python]
symbol = "🐍 "
format = 'via [${symbol}Python ${version}]($style) '
style = "bold yellow"

Por que isso ajuda você no curso?

Sinal Visual O que significa?
➜ (Verde) Tudo certo! Pode digitar seu próximo comando Git.
✗ (Vermelho) Atenção! O último comando deu erro. Verifique o que foi digitado.
🐍 Python Confirmação de que o ambiente de programação está ligado.
 main Mostra em qual "versão" do projeto você está trabalhando agora.
