Configuração Básica do Git

O comando git config permite ajustar configurações importantes do Git, como identidade do usuário, editor padrão, comportamento de cores e muito mais.

Configurar identidade


git config --global user.name "Seu Nome"
git config --global user.email "seuemail@example.com"

Essas informações serão usadas para identificar seus commits.

Configurar editor padrão


git config --global core.editor "code --wait"

Ver todas as configurações


git config --list

Configurações recomendadas

Aqui estão algumas configurações úteis para melhorar sua experiência com o Git: 


git config --global color.ui auto
git config --global pull.rebase false
git config --global init.defaultBranch main
